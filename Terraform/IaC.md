# Infrastructure as Code
- Using code we can deploy infrastructure resources in various cloud
- Simple human readable language

## Tools
- Ansible
- Terraform
- CloudFormation
- Docker

## Types of IAC
1. Configuration Management
    - Ansible
    - Puppet
    - Saltstack
2. Server Templating
    - Docker
    - Vagrant
    - Packer
3. Provisioning Tools
    - Terraform
    - CloudFormation
